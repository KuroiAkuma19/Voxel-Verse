# Voxel-Verse

A procedurally generated 3D voxel engine built with **React** and **Three.js**. This project features infinite terrain generation, dynamic biomes, and a functional day-night cycle, all optimized for the web.

## 🚀 Key Features

* **Infinite Procedural Terrain:** Real-time world generation using noise algorithms.
* **Three.js Rendering:** High-performance 3D graphics rendered directly in the browser.
* **Dynamic Environments:** Distinct biomes and a reactive day-night lighting system.
* **Optimized Mesh Generation:** Efficient handling of voxel data to maintain high frame rates.

## 🛠️ Built With

* **React** - Frontend framework.
* **Three.js** - 3D engine.
* **Vite** - Lightning-fast build tool and development server.
* **JavaScript/HTML/CSS** - Core web technologies.
